# Naming Convention

Suffix

    e   Central/Eastern European
    g   Greek
    r   Russian
    t   Turkish

Prefix

    c   Traditional Chinese (Big5)
    h   Korean (Hangul)
    j   Japanese (Shift-JIS)
    s   Simplified Chinese (GB2312)

Code Page

    874     Thai
    1255    Hebrew
    1256    Arabic
    1257    Baltic
